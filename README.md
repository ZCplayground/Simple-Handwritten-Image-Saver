# 简易手写体识别图片涂鸦工具

使用 PyQt 编写的简易工具，可以用鼠标画东西，可以保存为文件。

使用时需要修改 `scrawl.py` 中第 8 行的文件路径。

目前仅支持黑色画笔和保存、清空功能。
